Pug with Express.js
==

_Following this article :_

><https://www.sitepoint.com/a-beginners-guide-to-pug/>

_Config repo :_

```bash
mkdir pug-express
cd pug-express
npm init -y
npm i express
npm i -g json-server
npm i pug
npm i axios
```

_Running :_

```bash
json-server --watch db.json -p=3001
node app.js
```
